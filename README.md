


##  Install via CommandLine
Install the tools and start using it right away, its at user risk.
Its a simple tools that will help migrate SQL files into MYSQL database.


Step 1
```bash
 curl -sL https://raw.githubusercontent.com/miljimo/easymigration/refs/heads/master/install.sh | bash

```

Step 2

```bash
  obz help
```


To migrate the database you have to create a configuration files  in json which will be the input to the

```bash
 obz start -f [PATH]
 ```
